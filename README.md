#### Guardar cambios en el repositorio local (commit)
```bash
$ git add --all
$ git commit -m "readme.md descripcion"
```

#### Subir cambios locales al repositorio remoto (push)
```bash
git push
```


#### Acualizar la repo
```bash
git pull
```
